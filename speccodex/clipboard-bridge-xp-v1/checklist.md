- [ ] 机器配置支持共享剪贴板开关

- [ ] 宿主机可轮询系统剪贴板并推送文本
- [ ] XP 客户机工具可连接 `10.0.2.2:<port>`
- [ ] XP 客户机工具可读取并写入 `CF_UNICODETEXT`
- [ ] 双向同步已实现文本哈希防循环
- [ ] 断线可自动重连
- [ ] `Sanaka Tools ISO` 可生成并挂载
- [ ] `setup.exe` 可完成安装与开机自启
- [ ] 安装完成后重启可自动运行客户机工具
- [ ] 前端可显示连接状态与文本-only 说明
- [ ] 超长文本会被安全拒绝
- [ ] 非文本内容不会造成崩溃
