# Sanaka 未来计划

## 1. 增强功能工具链

- 完成 `Sanaka 增强功能程序` 的正式工具盘
- XP 优先落地共享剪贴板客户端
- 工具盘内放入：
  - 安装程序
  - 卸载程序
  - 开机自启后台程序
  - 配置文件
- 后续扩展到更多客户机系统

## 2. ISO 生成能力

- 让 Sanaka 自己具备生成 ISO 的能力，而不是每次都依赖外部手工处理
- 先服务于：
  - `Sanaka 增强功能工具镜像`
  - 将来的测试盘
  - 将来的驱动盘 / 附加工具盘
- 后续可以考虑开放成更通用的内部能力

## 3. 共享剪贴板

- 第一版只支持文本剪贴板
- XP 必须支持
- 后续再考虑：
  - 图片剪贴板
  - 文件剪贴板
  - 更细的状态展示
  - 更稳定的客户机安装体验

## 4. 虚拟机工具生态

- 除了剪贴板，还可以逐步扩展：
  - 宿主机 / 客户机信息同步
  - 更好的共享文件交换方式
  - 更完整的测试工具盘

## 5. 打包与分发

- `sanaka-tools.iso` 最终应进入正式打包资源
- 应避免发布包里按钮可见但资源缺失
- 未来要把：
  - macOS
  - Windows
  - Linux
  的增强功能资源一起纳入构建链

## 6. 长线方向

- Sanaka 不只是 QEMU 前端
- Sanaka 应逐步形成：
  - 虚拟机管理
  - 运行时集成
  - 工具盘
  - 客户机增强能力
  的完整产品闭环
